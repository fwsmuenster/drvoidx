Blocks

Mit der funktion lib:atblocklooking kann man einen befehl an der Position ausfürhen auf den der Spieler schaut.

function lib:atblocklooking
.mcfunction 
function lib:atblocklooking {c:"setblock ~ ~ ~ air"}

Bossbar

Mit function lib:bossbar/add kann man eine Bossbar hinzufügen. b: bossbar id, n: bossbar name, c: color (red, green, blue, yellow, purple, white)

function lib:bossbar/add
.mcfunction 
function lib:bossbar/add {b:"slime_boss",n:"SlimeBoss",c:"green"}

Functions (.mcfunction)

In Functions listest du normale Minecraft-Befehle zeilenweise auf. Sie werden im Spiel ohne Schrägstrich (/) geschrieben.

Pfad: data/mein_pack/functions/welcome.mcfunction
Minecraft Command 
tellraw @a {"text":"Willkommen auf dem Server!","color":"green"}
effect give @a minecraft:speed 10 1 true
give @p minecraft:cooked_beef 64
